# Portal Frontend - Authentication Gateway

A Vue.js-based authentication portal that serves as the primary entry point for students, teachers, and stakeholders accessing Avanti Fellows' educational platforms.

## What This Portal Does

**üîê Universal Authentication**
- **Sign-in/Sign-up flows** with ID, phone number, and date of birth verification
- **Multi-platform support** - seamlessly redirects to Quiz, Forms, Reports, Gurukul, and external platforms
- **Smart form handling** with dynamic dropdowns populated from database (schools, districts, colleges)
- **Session management** for live classes and assessments
- **Sessionless authentication** for direct platform access

**üè¢ Multi-Stakeholder Support**
- **Students**: Access to tests, assignments, and learning platforms
- **Teachers**: Broadcast sessions
- **Schools/Principals**: Broadcast sessions and reports

**üöÄ More Features**
- **OTP-based authentication** for secure phone verification
- **Basic A/B testing support** for feature experimentation
- **Two language support** (English/Hindi)
- **Responsive design** optimized for mobile devices
